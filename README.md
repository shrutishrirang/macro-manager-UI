# MacroManager Streamlit Frontend

A notebook-aesthetic Streamlit frontend for the PCOS-focused **MacroManager** nutrition tracking app.

## Key Features
- **PCOS Baseline Calibrator**: Personalized macronutrient goals based on health profiles.
- **Interactive SVG HUD**: Custom circular rings tracking daily Protein, Carbs, and Fat limits.
- **Multimodal Input**: Log meals via natural text, integrated voice/microphone, or photo/camera.
- **Today's Journal & Weekly Trends**: Summarizes daily intake and long-term macronutrient balances.
- **AI Memory System**: Keep track of persistent preferences, allergies, routines, and utensils.
